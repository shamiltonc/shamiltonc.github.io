For this enhancement, I focused on my RESTful API’s interaction with the Mongo database.

I improved my queries to get better results. The new and improved queries use the proper HTTP CRUD methods when querying the RESTful API and retrieving information from the database. To do this, I had to recreate all my curl queries for the proper CRUD/HTTP method:

POST: curl -d '{"company":Corona,"industry":"Medical"}' -H 'Content-Type: application/json' http://localhost:8080/createStock/

GET: curl -v http://localhost:8080/getStock?company=”Corona”

UPDATE: Query: curl -d '{"company":"Corona", "Volume":"True"}' -H 'Content-Type: application/json' -X PUT http://localhost:8080/updateStock

DELETE: curl -v -X DELETE http://localhost:8080/deleteStock?company=”Corona”
